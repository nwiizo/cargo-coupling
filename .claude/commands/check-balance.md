# Check Balance - バランススコア確認

このコマンドは、プロジェクトのカップリングバランススコアを素早く確認します。

## 使用方法

```
/check-balance [パス]
```

例:
```
/check-balance ./src
/check-balance .
```

## 実行内容

1. `cargo run -- coupling --summary [パス]` を実行
2. バランススコアとヘルスグレードを確認
3. 問題がある場合は簡潔なアドバイスを提供

## スコア解釈ガイド

| スコア | グレード | 状態 | アクション |
|--------|----------|------|-----------|
| 0.90-1.00 | A | 優秀 | 維持 |
| 0.80-0.89 | B | 良好 | 軽微な改善 |
| 0.60-0.79 | C | 許容 | 計画的改善 |
| 0.40-0.59 | D | 要改善 | 早期対応 |
